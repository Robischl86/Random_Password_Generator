## Usage
Once the application executes, the user will be prompted to enter a number of characters for the password, followed by a selection of character types: Upper-case letters, lower-case letters, numerals, and special characters. Once the selections are made, the user can click on the Generate Password button to generate a new string of characters based on the user's preferences.
